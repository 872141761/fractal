启动普通节点
------------------------------------------
普通节点不会开启挖矿工作。   
如果需要开启挖矿，请参考 [启动挖矿节点](#启动挖矿节点)

*如果你已经启动了gftl进程, 请先使用Ctrl+C将其停止。*
1. 准备工作目录。接下来所有步骤都将基于该工作目录。
    ```
    $ mkdir -p ~/fractal-test
    $ cd ~/fractal-test
    ```
2. 新建账户。  
    如果你的工作目录中已经有key文件了，请忽略本步骤。

    如果你从其他地方获得过key压缩包，拷贝它到工作目录并解压:
    ```
    $ mkdir -p data
    $ unzip key.zip -d data/
    ```
